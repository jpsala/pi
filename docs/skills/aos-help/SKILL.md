# OS Help

Mostrar el mapa corto de comandos AOS.

Fuente canonica: `docs/topics/os-quality.md`, seccion `Comandos Naturales`, y `docs/.generated/context-index.md` para recursos actuales.

## Respuesta

Listar comandos por intencion: continuar en esta sesion (`aos-sigamos`, `gol`), usar threads seguros (`aos-orquestar`, `aos-fanout`), guardar valor (`guardar sesion`), cortar contexto (`nueva sesion`, `nueva sesion con gol`), calidad OS (`realinear os`, `perfect os`), init/adopt/update, commit/push y comandos Pi si aplican. Mostrar `aos-checkpoint`, `cerrar sesion`, `continuar sesion` y `siguiente` solo como aliases.

No ejecutar cambios salvo que JP pida un comando concreto.
